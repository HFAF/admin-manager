# COMPONENTS

The components directory contains your Vue.js Components.
Nuxt.js doesn't supercharge these components.

**This directory is not required, you can delete it if you don't want to use it.**

## フォルダ構成について

```
components
├── profile（プロフィール）
│   ├── View.vue
│   └── Edit.vue
├── reservation（予約管理）
│   └── ***.vue
└── medical-chart（カルテ）
    └── ***.vue
```
